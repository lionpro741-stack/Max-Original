# 🚀 Roadmap: Telegram Lite Messenger

План развития мессенджера для создания лайт-версии в стиле Telegram.

---

## 🔥 Приоритетные функции (MVP)

| # | Функция | Описание |
|---|---------|----------|
| 1 | **Страница `/chats`** | Список всех чатов с последним сообщением и аватаром |
| 2 | **Статус "онлайн/офлайн"** | Показывать, когда пользователь был в сети |
| 3 | **Счётчик непрочитанных** | Бейджик с количеством новых сообщений |
| 4 | **Поиск чатов/контактов** | Поиск по имени пользователя |
| 5 | **Удаление сообщений** | Удалить своё сообщение (для себя/для всех) |

---

## 💬 Улучшения чата

## 📋 Рекомендуемый порядок реализации

```
Этап 1: /chats страница → Статус онлайн → Удаление сообщений
Этап 2: WebSocket → Галочки прочтения → Тёмная тема
Этап 3: Прикрепление файлов → Редактирование → Поиск
Этап 4: Групповые чаты → Боты → PWA
```
